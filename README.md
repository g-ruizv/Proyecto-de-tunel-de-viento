# Proyecto Túnel de Viento - Universidad de los Andes

Este repositorio contiene la documentación y los archivos fuente correspondientes al diseño, ensamblaje e implementación de un túnel de viento modular. El proyecto integra el desarrollo de hardware especializado para el control de módulos de ventilación y una interfaz digital para su operación remota.

## Descripción General

El sistema se compone de una estructura física controlada mediante placas electrónicas dedicadas y una plataforma de software que permite la gestión de los parámetros del túnel. Este repositorio ha sido estructurado para centralizar el código de la interfaz, el firmware de los controladores y la documentación técnica necesaria para su correcta ejecución y mantenimiento.

## Contenido del Repositorio

Para facilitar la comprensión y el uso del sistema, la información se ha dividido en secciones especializadas. Cada directorio cuenta con su propio documento descriptivo (READ.me) con instrucciones detalladas:

1. **Interfaz de Control:** Contiene el software necesario para ejecutar la aplicación de usuario.
2. **Controladores de Hardware:** Incluye el código fuente para las placas controladoras y la lógica de operación de los módulos.
3. **Recursos Técnicos:** Compendio de datasheets de componentes, diagramas de circuitos y manuales de implementación.

El objetivo de esta organización es garantizar una transferencia de conocimiento efectiva y proporcionar una base técnica sólida para futuras iteraciones o mantenimiento del túnel de viento.
